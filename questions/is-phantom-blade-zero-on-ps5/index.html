Is Phantom Blade Zero on PS5?

Yes. Phantom Blade Zero is officially listed for PlayStation 5 and is scheduled for October 29, 2026 at 02:00 UTC.

Official PS5 Confirmation

The PlayStation Store lists a PS5 version from S-GAME Publishing. It shows the game as a one-player title and currently offers wishlisting rather than purchase.

Listed PS5 Features

  • DualSense wireless controller vibration.
  • DualSense trigger effects.
  • PlayStation Game Help support.
  • Single-player support on the current listing.

What Is Not Confirmed Yet

The store does not yet specify performance and quality modes, target frame rates, download size, preload timing, trophy details, price, or edition contents. Avoid treating preview builds as final performance evidence.

Other Platforms

A Windows PC version is also confirmed, so the game is not PS5-only. No Xbox or PS4 edition has been announced.
