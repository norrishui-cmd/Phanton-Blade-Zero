Does Phantom Blade Zero Support Controller?

A controller-support page for PC, PS5, remapping, prompts, vibration, and comfort settings.

Short Answer

Controller support should be verified by official storefront tags and final menu testing. For a fast action game, controller comfort deserves its own page.

What to Verify

  • Supported controller types on PC.
  • Button prompt behavior.
  • Remapping and sensitivity options.
  • Vibration and DualSense-specific features if available.
  • Keyboard and mouse comparison.

Confirmed Controller Support

The PS5 version necessarily supports the DualSense controller, and its official page specifically lists vibration and trigger effects. Steam controller details have not yet been fully published. PC testing will verify wired and wireless DualSense behavior, Xbox-style pads, button prompts, remapping, haptics, and fallback input.

The release guide will record the platform, connection method, controller firmware, and game patch for every tested result, so basic input support is not confused with full haptics or adaptive-trigger support.
