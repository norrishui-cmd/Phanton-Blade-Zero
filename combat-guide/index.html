Confirmed Combat Baseline

A PlayStation hands-on preview describes three defensive choices: hold L1 to block, time a parry, or dodge for a counterattack opportunity. S-GAME has separately framed the game as fast action inspired by wuxia and 1990s kungfu cinema, with complex-looking move chains designed to require less button mashing than classic character-action games. That confirms the broad decision layer, but not final frame data, resource costs, invulnerability windows, or universal counters.

How to Use This Guide at Launch

Begin with the block guide, then compare the parry and dodge pages. Boss-specific pages will only prescribe a response after the release build has been tested.
