Block, Parry, and Dodge Confirmed in Phantom Blade Zero Hands-On

PlayStation's Summer Game Fest hands-on described holding L1 to block, while timed parries and dodges could create powerful counterattack opportunities.

What Is Confirmed

The preview is strong evidence for the core defensive choices, but it is not a final frame-data guide. It does not publish retail parry windows, dodge invulnerability, guard durability, chip damage, resource costs, difficulty modifiers, or a universal answer for every attack.

What It Means for Players

The combat hub will teach observation first: block a repeatable string, then test parry and dodge separately while keeping weapon and distance constant. Boss pages will recommend a response only after it is reproducible in the release build and labeled with platform, difficulty, and patch.
