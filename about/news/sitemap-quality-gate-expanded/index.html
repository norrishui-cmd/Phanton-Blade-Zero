Sitemap Quality Gate Now Covers SEO, Locales, and AdSense

The deployment quality gate has expanded beyond simple sitemap generation to catch technical and content issues before they reach production.

What Is Confirmed

Checks include unique titles, descriptions, self-canonicals, one H1, working internal links, valid JSON-LD, minimum content depth, matching localized language paths, required hreflang, exact sitemap membership, and AdSense ownership code. German uses a word threshold; Japanese uses a character-based threshold appropriate for its writing system.

What It Means for Players

A page does not enter the sitemap merely because a file exists. It must be indexable and pass the audit. Future News, FAQ, weapon, boss, quest, and localization batches will run generation and audit together, with zero blocking errors required before packaging.

This update is based on Generated sitemap index. It separates confirmed statements from inferences and records the publication date so later announcements or retail testing can replace outdated information.
