Phantom Blade Zero Items and Upgrade Materials

A launch-ready hub for consumables, upgrade materials, rewards, and equipment resources without inventing item names before release.

What This Board Should Include

For an action RPG, item pages should be practical: where to find the item, what it upgrades, whether it is limited, and whether using it affects a build or quest. Until item names are verified, this hub should define the structure rather than publish fake item entries.

Item Page Template

  • Exact item name and category.
  • First available location and repeatable sources.
  • Used for weapons, Phantom Edges, healing, quests, or progression.
  • Whether the item is missable, farmable, or limited.
  • Related build, boss, and route links.

URL Expansion

Verified item pages can scale safely after launch because each item has a distinct location/use question. The quality gate is simple: no exact name and tested use, no indexable item URL.
