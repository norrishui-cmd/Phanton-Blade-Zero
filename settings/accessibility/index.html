Accessibility

Phantom Blade Zero Accessibility Settings

A player-support page for accessibility and comfort settings that should be verified from the final game menus.

Why This Page Matters

Fast melee action games can be demanding for reaction time, visual clarity, camera comfort, and repeated inputs. Accessibility coverage should help players decide whether the game can be tuned to their needs.

Launch Verification Checklist

  • Subtitles, subtitle size, and speaker labels.
  • Brightness, contrast, HUD, and visual effect options.
  • Camera shake, motion blur, and sensitivity options.
  • Remapping, hold/toggle behavior, and input assists if present.
  • Difficulty modes or assist features if offered.
