Phantom Blade Zero Release Date Change Tracker

A transparency page for tracking release date changes, conflicting old dates, and countdown updates.

Current Wiki Date

This wiki currently tracks October 29, 2026. Some older pages around the web may still show September 9, 2026, so the release hub should be rechecked against official and storefront sources before deployment and before GSC submission.

What Changes When the Date Changes

  • Homepage countdown and release facts.
  • Release page, FAQ, and time-zone page.
  • Sitemap lastmod date.
  • Update notes and source policy.

Editorial Rule

If official pages conflict with third-party reporting, mark the page as "current tracked date" and add a verification note until the conflict resolves.

Current Official Timing

PlayStation lists October 29, 2026 at 02:00 UTC. Steam shows October 28, 2026, which is consistent with the same global moment appearing on the previous calendar date in some time zones. This is a storefront display difference, not evidence of two separate launches.

How a Real Date Change Is Confirmed

The page will update only when S-GAME, PlayStation, or Steam changes the published timing. A countdown glitch, retailer placeholder, or social rumor is not enough. If the date moves, the release hub, time-zone table, homepage, FAQ, schema, sitemap lastmod values, and GSC priority URLs should be refreshed together.
