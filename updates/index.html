Phantom Blade Zero Updates and Patch Notes

A hub for official news, release-date changes, trailer updates, demo information, patch notes, and guide revision history.

Update Coverage Rules

News and patch pages should cite official sources, record the date of the change, and explain which guide pages were affected. This keeps the wiki reliable as release details, combat terms, and platform information evolve.

Future URL Types

  • Official trailer update pages.
  • Release date and platform update pages.
  • Demo announcement and demo patch pages.
  • Post-launch balance patch pages.
  • Guide changelog pages for major revisions.

Internal Linking

Every update should link to the affected hub: release, platforms, combat, weapons, bosses, maps, or settings.

What Counts as an Update

Tracked changes include official release timing, platform listings, editions, demos, requirements, feature confirmations, trailers, State of Play details, and post-launch patches. Rumors and unsourced screenshots are not added to the timeline as facts. A changed storefront is dated and linked so readers can distinguish a new announcement from an editorial rewrite.
