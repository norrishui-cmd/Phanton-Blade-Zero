Steam Recommends a Controller for Phantom Blade Zero

Steam currently shows full controller support and states that the developers recommend using a controller.

What Is Confirmed

The storefront references Xbox controllers, PlayStation controllers, and Steam Input, but the final behavior of wireless DualSense features on PC, glyph switching, rebinding, vibration, adaptive triggers, and mixed keyboard-controller input still needs retail testing. PS5 separately confirms DualSense vibration and trigger effects.

What It Means for Players

Beginners should plan to have a controller available, but should not assume every PC pad receives identical haptics. The launch guide will test wired and wireless connections, button prompts, remapping, dead zones, Steam Input profiles, and fallback behavior, recording the game patch and controller firmware.

This update is based on Official Steam feature list. It separates confirmed statements from inferences and records the publication date so later announcements or retail testing can replace outdated information.
