Schema Markup Plan

A structured data roadmap for making guide clusters easier for search engines to understand.

Current Schema

The homepage includes WebSite and VideoGame schema. The FAQ page includes FAQPage schema.

Next Schema Types

  • BreadcrumbList on hub and child pages.
  • ItemList on hub pages such as questions, guides, comparisons, templates, weapons, and settings.
  • Article schema on verified guide pages after launch.
  • HowTo only for step-by-step verified guides, not speculative pages.

Quality Rule

Schema should describe visible page content. Do not add HowTo or review-style markup to placeholder or framework pages.
