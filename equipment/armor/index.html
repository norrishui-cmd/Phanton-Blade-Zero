Phantom Blade Zero Armor

Armor is an officially named reward and character-progression category, but no complete retail armor list, stat table, set-bonus system, or upgrade path has been published.

Official status

S-GAME's PlayStation introduction says powerful beings reward players with weapons, armor, artifacts, and skills. That confirms armor exists as a meaningful progression category without proving a conventional head, chest, hands, and legs slot structure.

What remains unknown

Slot count, defense formulas, weight, appearance changes, rarity colors, set bonuses, upgrade materials, transmog, and whether armor changes mobility are not established by the current sources. These details require the release build or a focused official showcase.

How armor pages will help

Verified entries will lead with location and acquisition requirements, then show base effects, upgrade changes, build fit, missable status, and patch version. This prevents generic pages that merely repeat an item description.

Source and update status

This page is grounded in the official source. It distinguishes published facts from details awaiting the dedicated State of Play or retail build. Rumors, unofficial names, and trailer-only assumptions are excluded.
