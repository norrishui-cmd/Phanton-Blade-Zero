Phantom Blade Zero Dodge Guide

A movement-first defensive guide for learning when dodging is safer than blocking or parrying.

Dodge as Positioning

In fast action combat, dodging is not only an invulnerability question. It is also a spacing tool: moving around large swings, staying near punish windows, leaving dangerous ground effects, and keeping the camera under control.

Training Path

Start with beginner guide, compare parry and block, then use boss pages for real examples.

Public Evidence

A PlayStation hands-on preview explicitly lists dodging alongside blocking and parrying and says a dodge can set up a devastating counterattack. It does not publish invulnerability frames, stamina cost, distance values, or a guarantee that every attack can be dodged in the same way. Those values remain launch-test items.

First Tests to Run

Test neutral and directional dodges at close and medium range against one repeatable attack. Note whether the success comes from invulnerability, moving outside the hitbox, or both. Then check recovery and camera position before recommending a punish. Boss guides will link here only after a response is repeatable.
