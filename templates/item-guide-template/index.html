Item Guide Template

A quality gate for future item, material, reward, and collectible pages.

Publish Gate

Publish an individual item page only after the exact item name, location, and use are verified. Avoid one-line item stubs.

Required Sections

  • Exact item name and category.
  • Location, route, and map link.
  • Use case: upgrade, quest, healing, progression, or collectible.
  • Farmability or limited status.
  • Related weapon, Phantom Edge, quest, or achievement.

Thin Content Avoidance

If an item has only a name and no verified use, keep it in the parent hub until more data is available.
